# Supported Platforms

## Supported first

- Ubuntu 22.04/24.04 on WSL2.
- Ubuntu/Debian-like Linux with `apt`, best effort.

## Expected tools

- Bash 4+.
- Python 3 for JSON escaping in `linux-doctor.sh` structured output.
- `apt-get`/`dpkg` for package checks.
- PowerShell on Windows for WSL export/install helpers.

## Not supported yet

- Fedora/RHEL package management.
- Arch package management.
- macOS Homebrew workflows.
- Non-WSL distro migration automation.
